## Setup Instructions

Install dependencies:

```bash
npm install
```

Setup ENV file

```bash
cp .env.example .env
```

Build the application:

```bash
npm run build
```

Run the server:

```bash
npm start
```

Open the application in the browser:

http://localhost:3000

## Libraries and Patterns

`shadcn` - for a number of accessible and visually appealing components

`tailwind` - for normalized functional css

`msw` - for mock users api

`react-hook-form` for form management

`zod` -  for for data validation

`playwright` - for browser automation testing
